# WordPress Theme Customization Demo (Astra)

A clean, performance-focused WordPress demo built in LocalWP using the Astra theme and the block editor.

## Overview
This project showcases intentional theme customization and responsive layout decisions without relying on heavy page builders or unnecessary plugins. The focus is on clean structure, typography, spacing, and maintainability across desktop and mobile.

## Key Features
- Responsive layout built with container blocks (Constrained / Flex where appropriate)
- Hero section tuned using viewport height (vh) for better scaling
- Consistent typography and spacing across breakpoints
- Clean navigation and calls-to-action
- Performance-conscious approach (minimal bloat)
