# WordPress Theme Customization Demo (Astra)

A clean, performance-focused WordPress demo built locally using LocalWP and the Astra theme.

## Overview

This project demonstrates intentional theme customization using the native WordPress block editor without relying on heavy page builders or unnecessary plugins.

The focus was on:

- Clean layout structure
- Performance-first decisions
- Responsive design
- Maintainable theme customization
- Clear visual hierarchy and spacing

---

## Key Features

- Customized Astra theme layout
- Responsive hero section (desktop + mobile optimized)
- Structured content containers
- Clean typography adjustments
- Performance-conscious design decisions
- Minimal plugin reliance
